All Commands

!channelonly {cmd name} {Channel tag or "none"} - Makes a command work in some channels.
!command {enable / disable} {cmd name} - Enableds or Disables a command for guild
!language {lang} - Displays or sets the language for this Discord server
!prefix {new prefix} - Displays or sets the prefix for the current guild
!requiredrole {cmd name} {"none" or role tag/id} - Specifies what role each command requires.
!setlevels - set new level-log channel
!setlogs - set new logs channel
!setmembercount {all | users | bots} {channel ID} - set new Member Counter channel
!setpunish - set new punish-log channel
!setverification {emoji} {role id} - set new verification channel
!setwelcome - set new welcome and exit channel
!rr {emoji} {role tag or ID} {some text} - Add Reaction to Reaction Roles Message
!rrmsg {channel tag} {text that will be visible} - Reaction Roles Message

!help - see all bot comands
